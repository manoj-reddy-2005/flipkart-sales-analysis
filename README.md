# Flipkart Sales Analysis (SQL Project)

This project analyzes Flipkart sales data using MySQL.

## Tools Used
MySQL
Power BI
Excel

## Analysis Done

- Total Revenue
- Total Profit
- Profit Margin
- Sales by Category
- Sales by Subcategory
- Top Cities by Sales
- Monthly Revenue Trend
- Sales by Payment Mode
- Top Customers
- Running Total Sales

## SQL Concepts Used

Joins  
Group By  
Window Functions  
Ranking  
Aggregate Functions

## Dashboard
Power BI dashboard created to visualize revenue, profit, categories and states.
